Pingpong Labs

# Introduction

Pingpong Sky is a PHP component made specifically for Laravel. Pingpong Sky includes some ready-made components such as Modules, Menus and Widgets.

# Features

  • Menus
  • Modules
  • Widget
  • Shortcode
  • Presenters
  • Themes
  • Trusty
  • Generators
